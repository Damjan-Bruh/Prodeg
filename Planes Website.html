 What is Planes?
In this game, you are plane that has to dodge lasers that are comming towoards you. You need to move up and down to control plane. There are to lasers, so you need to be careful. Controls are: w and s for up and down.
About programs you need
Prodeg Enemy Planes is a NON executable file, so you will need libraries and programs installed. Firstly, you will need Python 3.12 or higher, because it is a python file(Click link named Python to go to python webpage). Secondly, you will need pygame installed, which you can install by opening cmd and typing 'pip install pygame'(You will need python for this to work). And, finaly, you will need to download Prodeg Planes by clicking link named Download. You will also need to download IDLE python. Also, you will need to download file from link named BG Download. You Should be able to run Planes.
