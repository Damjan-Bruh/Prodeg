Prodeg Laser Atack

 What is Laser Atack?
In this game, you are a gun in space, that shoots in four directions(up, down, left, right). You need to shoot the bullets that are comming. They are comming out of all four directions and each direction has three bullets, but you also have three bullets for each direction. Controls are: w, a, s, d for left, right, up and down, t, f, g, h for left, right, up and down and i, j, k, l for left, right, up and down.
About programs you need
Prodeg Laser Atack is a NON executable file, so you will need libraries and programs installed. Firstly, you will need Python 3.12 or higher, because it is a python file(Click link named Python to go to python webpage). Secondly, you will need pygame installed, which you can install by opening cmd and typing 'pip install pygame'(You will need python for this to work). And, finaly, you will need to download Prodeg Laser Atack by clicking link named Download. You will also need to download IDLE python. Also, you will need to download file from link named BG Download and G Download. You Should be able to run Laser Atack.
