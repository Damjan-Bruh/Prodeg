Prodeg Guy And Arrow

 What is Guy And Arrow?
In this game, you are a guy that has to jump over the arrows. That is the whole game... Control is space for jump.
About programs you need
Prodeg Guy And Arrow is a NON executable file, so you will need libraries and programs installed. Firstly, you will need Python 3.12 or higher, because it is a python file(Click link named Python to go to python webpage). Secondly, you will need pygame installed, which you can install by opening cmd and typing 'pip install pygame'(You will need python for this to work). And, finaly, you will need to download Prodeg Guy And Arrow by clicking link named Download. You will also need to download IDLE python. Also, you will need to download file from link named BG Download. You Should be able to run Guy And Arrow.
