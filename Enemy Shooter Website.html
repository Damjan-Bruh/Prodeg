What is Enemy Shooter?
In this game, you are a gun that is pointing in the sky. But, randomly, some humans pop out and you need to shoot them. But you need to be fast, because your time is ticking and if you are not fast enough, you lose. Controls are: a and d for left and right, left arrow and right arrow for faster left and right and s is for shooting.
About programs you need.
Prodeg Enemy Shooter is a NON executable file, so you will need libraries and programs installed. Firstly, you will need Python 3.12 or higher, because it is a python file(Click link named Python to go to python webpage). Secondly, you will need pygame installed, which you can install by opening cmd and typing 'pip install pygame'(You will need python for this to work). And, finaly, you will need to download Prodeg Enemy Shooter by clicking link named Download. Also download IDLE python to run this. You should be able to run Enemy Shooter.
